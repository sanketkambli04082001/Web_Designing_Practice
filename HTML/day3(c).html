Python
Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented and functional programming.
Django
Django is a free and open-source, Python-based web framework that runs on a web server. It follows the modeltemplateviews architectural pattern. It is maintained by the Django Software Foundation, an independent organization established in the US as a 501 non-profit.
